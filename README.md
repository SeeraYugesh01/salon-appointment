# Salon Appointment Scheduler

A Bash application that allows customers to schedule salon appointments using a PostgreSQL database.

## Features

- View available salon services
- Register new customers
- Find existing customers by phone number
- Book appointments

## Technologies Used

- Bash
- PostgreSQL
- Git

## Run

```bash
./salon.sh
```

This project was completed as part of the freeCodeCamp Relational Database Certification.
